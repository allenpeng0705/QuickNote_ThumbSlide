Introduction:
 ThumbSlide is one tool for recording notes, writing EMail and SMS. It provides the simple logic and good inputting experiences and make the inputting much easier on iPhone.

Quick Input:
 You can use the quick keyboard to input and it provides the best error correction feature and you can input easy and fast on the iPhone. Quick Keyboard doesn't care about how you input (tap, trace, or together), you can get the correct word after you press "Space". In a word, the space key is very very important on the Quick Keyboard. After you finish one word, you should press the space key to get the correct word.

Smart Select:
 Quick Keyboard will provide the candidates list for selection when inputting. You can replace the inputting word by selecting other word from candidates list. The second word of the candidates list is the recommended word from quick keyboard and it will be sent to the text box as default. The first word of candidates list is the exact keyboard input, and you can get the word which Quick Keyboard cannot recognize. Another important feature is recapturing. When you select one word, we will fill the selection list again.

Auto Capitalization:
 Quick Keyboard will the auto-capitalization feature. You needn't to care about the capitalization and needn't to press "Shift" key when inputting.

Send Mail:
 After you finish one note, you can press "Done" to display the toolbar. You can select "Mail" from the toolbar, and the content will be send to the native EMail application.

Send SMS
 After you finish one note, you can press "Done" to display the toolbar. You can select "SMS" from the toolbar, and the content will be send to the native Message application.

Create:
 You can create new notes by selecting the + button in the upper right hand side of the main list of notes. This will create a new blank note for editing.

Edit:
 You can edit notes by selecting a note from the list on the main list page. This will bring your existing note up in the editor. You can edit the note and save your changes.

Delete:
 You can delete notes by selecting the edit button in the upper left corner on the main list page. This will enable the standard list editing function to allow you to delete notes.

Mail/Backup:
 You can mail notes by selecting the "Mail" button in the lower left corner on the notes view page. This will bring up an action list that lets you select Mail Note from which point you can address and send your note.

Sort:
 You can sort all the notes based on date or the title of notes by selecting the "Sort" button from the tool bar of note lists.

Search:
 You can search the note based on the keyword by selecting the "Search" button from the toolbar of note lists.
